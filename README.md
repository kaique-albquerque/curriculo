# Kaique Albuquerque | Curriculo Online

Curriculo online desenvolvido com React, TypeScript e Vite para apresentar experiencia profissional, formacao, competencias, certificados e canais de contato em uma interface moderna e responsiva.

## Tecnologias

- React
- TypeScript
- Vite
- CSS puro

## Funcionalidades

- Layout responsivo
- Tema claro, escuro e automatico
- Secoes de experiencia, formacao, competencias e certificados
- Links rapidos para LinkedIn, GitHub e WhatsApp
- Estrutura simples para editar os dados do curriculo em um unico arquivo

## Estrutura principal

- [src/App.tsx](./src/App.tsx): composicao da interface
- [src/data.ts](./src/data.ts): dados do curriculo
- [src/styles.css](./src/styles.css): estilos globais

## Como rodar localmente

```bash
npm install
npm run dev
```

## Build de producao

```bash
npm run build
```

## Personalizacao

Para atualizar o conteudo do curriculo, edite:

- `src/data.ts`

Para ajustar layout, cores e responsividade, edite:

- `src/styles.css`
